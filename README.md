#Events Exercise

## Description
Page dynamically creates divs and adds div content.  Uses ID's to 'show' and 'hide' content depending on which link a user clicks.

## How to run
In your terminal run:
```
http-server -p 9999
```
then navigate to http://localhost:9999 in your browser.
